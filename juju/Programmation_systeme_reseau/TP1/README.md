# TP1 - Multiprocessing

## Exercice 1

Compilation :
```
gcc -Wall compte_arg.c -o exec
```
Exécution :
```
./exec <liste de nombre>
```
Exemple :
```
./exec 1 2 3
```

## Exercice 2 et 3

Compilation :
```
gcc -Wall cp.c -o exec
```
```
gcc -Wall cp_reverse.c -o exec
```
Exécution :
```
./exec <fichier1> <fichier2>
```
Exemple :
```
./exec fichier1.txt fichier2.txt
```
